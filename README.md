## Установка проекта
Перейти в корень проекта, запустить консоль и прописать

```sh
npm install
```

### Запуск тест-сервера
Из корня проекта в консоле прописать

```sh
npm run dev
```

### Сборка проекта
Из корня проекта в консоле прописать

```sh
npm run build
```
Собранный проект появится в папке dist
